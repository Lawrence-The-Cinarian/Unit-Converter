# Unit-Converter
Conversion from One Unit to another 

# A Unit Converter 
with these characteristics 
(1) GRAMS TO KILOGRAMS.                   (2) KILOGRAMS TO GRAMS.          
(3) METRES TO KILOMETRES.              (4) KILOMETRES TO METRES.          
(5) CENTIMETRES TO MILLIMETRES.    
(6) MILLIMETRES TO CENTIMETRES.         
(7) MINUTES TO SECONDS.                 
(8) SECONDS TO MINUTES.                 
(9) CELSIUS TO KELVIN.                  
(10) KELVIN TO CELSIUS.

More will still come in later 

# The Unit Converter as a C script 
it a basic C Script aimed for helping me build my foundation on systems programming
